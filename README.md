SnappShop Interview Task: A Laravel-based small app.

## Usage

Start the Docker containers using Laravel Sail:

```
./vendor/bin/sail up
```

Access the application in your browser at http://localhost.

## Endpoints
- POST /api/transaction/transfer
- GET /api/user/top
